# CS-Xmind-Note
## 说明
  用于学习CS基础课程，知识点来源完成后会一一列举，目前的目标是：
  
    数据结构和算法--《大话数据结构》、《王道考研》、《算法》、各大博客论坛（到时候会备注地址）
  
    计算机网络   --《计算机网络-自顶向下方法》、《王道考研》、《TCP/IP协议：卷一》
  
    计算机组成原理-- 目前没有目标,有好书的可以推荐一下，谢谢。
  
    操作系统     -- 同上
  
    编译原理     -- 《编译原理》（龙书）
  
    设计模式     -- 《Header First 设计模式》
